# Advent of Code in Scala

Advent of Code ( https://adventofcode.com ) 2015-2020 solutions using pure-functional scala

Prerequisites
-----

* mill

Build
-----

    $ mill app.compile

Tests
-----

    $ mill problems.test
    $ mill app.testAll

Solve Problem
-----

    $ mill app.problem 2015_01_1
    $ mill app.problem 2015_01_2
